# 字幕時間軸校對工具 PRO MAX

這是一個純前端字幕校對工具，可直接開啟 `index.html` 使用，也可以部署到 GitHub Pages。

## 功能

- 載入音訊：mp3 / wav / m4a 等瀏覽器支援格式
- 載入 SRT 字幕
- 整段波形顯示
- 時間軸刻度
- 播放指針
- 播放時自動高亮目前字幕
- 字幕片段可拖曳調整時間
- 字幕列表可直接修改開始時間、結束時間、文字
- 整體時間偏移
- 基礎自動延遲偵測
- 匯出修正後 SRT

## 快捷鍵

| 快捷鍵 | 功能 |
|---|---|
| Space | 播放 / 暫停 |
| ← / → | 前後跳 0.5 秒 |
| A / D | 選取字幕 -0.1 / +0.1 秒 |
| Delete | 刪除選取字幕 |

## 使用方式

1. 解壓縮 ZIP
2. 打開 `index.html`
3. 載入音訊
4. 載入 `.srt`
5. 校正字幕
6. 點擊「匯出 SRT」

## 部署到 GitHub Pages

把整個資料夾內容上傳到 GitHub repository，並在 Settings → Pages 啟用即可。
