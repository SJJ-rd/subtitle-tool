# 字幕時間軸校對工具 TRUE FINAL

這是真正完整版，可直接部署到 GitHub Pages。

## 功能

- 載入音訊
- 載入 SRT
- 整段波形顯示
- 播放指針
- 播放時自動高亮目前字幕
- 字幕片段中間拖曳移動
- 字幕片段左側拉伸：調整開始時間
- 字幕片段右側拉伸：調整結束時間
- 時間軸縮放
- 整體偏移
- 基礎自動延遲偵測
- 字幕列表直接編輯時間與文字
- 匯出修正後 SRT

## 快捷鍵

- Space：播放 / 暫停
- ← / →：前後跳 0.5 秒
- A / D：選取字幕前後微調 0.1 秒
- Delete：刪除選取字幕

## 使用方式

1. 解壓縮 ZIP
2. 打開 index.html
3. 載入音訊
4. 載入 SRT
5. 拖曳或拉伸字幕片段
6. 匯出 SRT

## GitHub Pages

把全部檔案上傳到 repository 根目錄，開啟 Pages 即可。
